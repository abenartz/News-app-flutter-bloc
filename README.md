# News-app-flutter-bloc

A Flutter sample app for using Bloc v8 and Dio.

## Main features
- Bloc v8 state managment (MVI arcitecture).
- Repository pattern.
- Dio http client package.
- Handle pagination using scroll controller.
- Bottom navigation.
- MaterialPageRoute.
